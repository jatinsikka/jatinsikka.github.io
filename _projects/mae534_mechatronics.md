Final project for MAE 534 - Mechatronics Design course, demonstrating integration of mechanical, electrical, and software systems.

## Overview

This project represents a comprehensive mechatronics design that combines mechanical components, electronic systems, and embedded software to create a functional mechatronic system.

## Project Components

- **Mechanical Design**: CAD modeling and fabrication
- **Electronics**: Circuit design and sensor integration
- **Software**: Embedded programming and control algorithms
- **Integration**: System-level integration and testing
